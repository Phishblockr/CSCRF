# CSCRF
Project Overview
This repository hosts the first-of-its-kind technical crosswalk between the SEBI Cybersecurity and Cyber Resilience Framework (CSCRF) and the CIS Critical Security Controls (v8.1).

In the wake of the 2024-2025 regulatory updates in the Indian financial sector, organizations are facing a "translation gap" between high-level regulatory mandates and actionable technical controls. 

This project bridges that gap, providing GRC professionals, CISOs, and Auditors with a roadmap to achieve SEBI compliance using the world-class CIS framework.

Why This Matters
Operationalizes Regulation: Translates abstract SEBI clauses into specific CIS Safeguards.
Audit-Ready: Includes logic for evidence collection, significantly reducing audit prep time.
Global Alignment: Allows Indian financial institutions to demonstrate compliance using a globally recognized security standard, facilitating international business and trust.

The Framework Structure
The mapping focuses on high-impact domains including:
Identity & Access Management (PR.AA)
Data Protection & Encryption (PR.DS)
Vulnerability Management & VAPT (Part IV)
API & Application Security (Annexure-G)
Supply Chain Risk Management (ID.SCM)
